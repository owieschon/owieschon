# Owen Schoeniger

<!-- clean-docs:purpose -->
I was a teacher before I was anything else, and curiosity did the rest. Teaching pulled me into 3D printing, 3D printing pulled me through manufacturing and industrial tech, and the road ran all the way to working with propulsion engineers and materials science PhDs at SpaceX, where machine malfunctions and miscalibrations had costly, real-world consequences. These days I like to build ML and agents that help people. Deterministic cores, inspectable evidence, and agents that can propose or verify without silently acquiring authority.
